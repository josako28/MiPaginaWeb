# MISLATAZON

MISLATAZON es una plataforma dedicada a vender cajas sorpresa de productos seleccionados de los palets de Amazon. Cada caja está diseñada para ofrecer un valor significativamente mayor al precio pagado, proporcionando una emocionante oportunidad para pequeños emprendedores y revendedores.

## Cómo Ejecutar

1. Clona este repositorio.
2. Abre `index.html` en tu navegador para ver la página principal.
3. Utiliza los enlaces de navegación para moverte entre las diferentes secciones del sitio web.

## Descripción

MISLATAZON ofrece cajas de productos de Amazon a precios accesibles, asegurando que el valor de los productos dentro de cada caja siempre supere el precio de la misma. También ofrecemos productos sueltos para aquellos que prefieren comprar artículos específicos.
